# 🌱 DreamSeed AI Tools

> DreamSeed 种梦计划参赛作品 — 一套轻量级 AI 工具集合

## ✨ 项目简介

这是一个面向普通用户的 AI 工具集合，提供简单实用的 AI 辅助功能，无需复杂配置即可使用。

### 🚀 核心功能

- **📝 智能文本优化** — 一键润色、改写、翻译文本
- **🎨 Prompt 生成器** — 根据需求自动生成高质量 AI 绘画提示词
- **📊 数据分析助手** — 简单数据可视化和分析
- **🔧 代码片段生成** — 根据描述生成常用代码片段

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **后端**: Python Flask
- **AI 接口**: 兼容 OpenAI API 格式（支持自定义端点）
- **部署**: 轻量级，支持本地运行或 VPS 部署

## 📦 快速开始

```bash
# 克隆项目
git clone https://github.com/youyhaa/dreamseed-ai-tools.git
cd dreamseed-ai-tools

# 安装依赖
pip install -r requirements.txt

# 启动服务
python app.py
```

访问 `http://localhost:5000` 即可使用。
